# Loop in loop

## 2중 for문

<hr/>

```java

	public static void main(String[] args) {
		
		int num = 1;
		
		for(int i = 0; i < 3; i++) {	  // 세로 줄을 결정짓는 요소
			for(int j = 0; j < 4; j++) {  // 가로 칸을 결정짓는 요소
				System.out.printf("(%d, %d) : %2d\t", i, j, num);	
				num++;
			}
			System.out.println();
		}
		System.out.println("끝");

		0,0 0,1 0,2 0,3
		1,0 1,1 1,1 1,3
		2,0 2,1 2,2 2,3 이 출력됩니다.
		중첩된 for문은 다수의 값을 2개 이상의 기준으로 나누어서 처리하고 싶을 때 사용합니다.
		배열 혹은 컬렉션과 연계하여 사용하는 경우가 단독으로 사용하는 경우보다 많습니다.
	}

```

## 로딩창의 원리

```java

	public static void main(String[] args) {
			
			Scanner sc = new Scanner(System.in);
			int length = 50;	// 길이를 50으로 정해둡니다.
			int num;
			
			System.out.print("정수를 50 이하로 입력 : ");
			num = Integer.parseInt(sc.nextLine());
			
			System.out.print("[");
			for(int j = 0; j < num; j++) {	
				System.out.print("#");
			}
			for(int j = 0; j < length - num; j++) {	 
				System.out.print("_");
			}
			System.out.print("]");
			sc.close();

			입력받은 정수만큼 #을 증가시킵니다.
			총 길이 - 입력값만큼 _로 채웁니다.
			

	}
```

## 로딩창 증가의 원리

```java

	public static void main(String[] args) throws Exception {
		Scanner sc = new Scanner(System.in);
		int length = 50;
		int num;
		
		System.out.print("정수 입력 (%): ");
		num = Integer.parseInt(sc.nextLine());
		
		for(int i = 0; i <= num; i++) {			
			System.out.println("\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n");
			System.out.print("[");
			for(int j = 0; j < length; j++) {	
				if(j == length / 2) {		
					System.out.printf(" %3d%% ", i);
				}
				else if(j < i / 2) {	
					System.out.print("#");
				}
				else {			
					System.out.print(" ");
				}
			}
			System.out.println("]");
			Thread.sleep(100);
			Runtime.getRuntime().exec("cmd /c cls");	// 운영체제에게 직접 명령을 전달
		}
		sc.close();


		입력받은 수만큼 \n을 추가해줍니다. %가 증가하는 만큼 공백을 줘야 자바 콘솔에서 로딩창처럼
		보이기 때문입니다.  
		length 값까지 j를 증가시키며 내부 반복문을 실행합니다.
		현재위치(j)가 length의 중앙일 경우 퍼센트 값을 출력합니다.
		현재 i 값에 해당하는 퍼센트 값을 출력합니다.
		현재 위치(j)가 입력받은 값보다 작은 구간일 경우 # 문자를 출력합니다.
		입력받은 값보다 크거나 같은 구간일 경우 공백을 출력합니다.
		0.1초의 딜레이를 주고 출력합니다.
	}
```

## 구구단

```java

	System.out.println("=== 구구단 ===\n");
		for(int i = 2; i < 10; i++) {	
			for(int j = 1; j < 10; j++) {	
				System.out.printf("%d X %d = %2d\n", i, j, i * j);
			}
			System.out.println();

			2부터 9까지 증가시킵니다 (단)
			1부터 9까지 증가시킵니다. (* n)
			값을 출력합니다. i x j = i*j ex_ 2 x 2 = 4
		}

```

## 구구단 정렬하기

```java
	public static void main(String[] args) {
		
		for(int j = 1; j < 10; j++) {	
			for(int i = 2; i < 10; i++) {		
				System.out.printf("%d X %d = %2d    ", i, j, i * j);
			}
			System.out.println();
		}
		System.out.println();
		
		1부터 9까지 증가시킵니다. (세로)
		2부터 9까지 증가시킵니다. (가로)
		출력 값 예시 _
		2x1 3x1 4x1 ~~~
		2x2 3x2 4x2
		2x3 3x3 4x3
		~~~

		for(int j = 2; j < 10; j++) {
			for(int i = 1; i < 10; i++) {
				System.out.printf("%d X %d = %2d    ", j, i, j * i);
			}
			System.out.println();
		}
		System.out.println();

		위의 예시와 반대로 출력하는 숫자의 위치와 방향이 달라지게 됩니다.
		첫 번째 예시에서는 한 줄에 9개의 곱셈 결과를 출력하면서 가로 방향으로 출력되고,
		두 번째 예시에서는 한 줄에 8개의 곱셈 결과를 출력하면서 세로 방향으로 출력되기 때문입니다.
		출력 값 예시 _
		2x1 2x2 2x3 2x4
		3x1 3x2
		4x1     4x3
		5x1         5x4
	}

```
